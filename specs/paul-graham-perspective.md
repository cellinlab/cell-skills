# Paul Graham 视角 Skill 设计文档

## 1. 目标概述

这个 Skill 用于把 Paul Graham 的创业、写作、产品与独立思考框架，整理成一个可复用的顾问型视角 Skill。

目标不是模仿 PG 的 essay 口气，而是让用户在遇到以下问题时，能够调用他稳定反复出现的判断镜片：

- 写作如何帮助思考
- 创业问题应该先看人、问题还是产品
- 怎样判断一个机会是否值得长期投入
- 怎样避免被主流共识拖着走

## 2. 设计来源

核心明显来自 `nuwa-skill` 仓库中的 `paul-graham-perspective`，但会做 Cell 风格改造。

保留的部分：

- 5 个高密度核心心智模型
- 创业 / 写作 / 创始人判断等高频启发式
- 表达 DNA 与诚实边界
- 对其内部矛盾与盲区的保留

改造的部分：

- 从角色扮演改成 advisor-first
- 拆成渐进式结构，避免一个巨型 SKILL.md
- 更强调“这个镜片适合解决什么，不适合解决什么”

## 3. 技能定位

它是一个“创业 / 写作 / 产品 / 独立思考”视角 Skill。

它不是：

- Paul Graham 语录收纳箱
- 英文 essay 模仿器
- YC 战术手册

## 4. 适用场景

- 用 PG 的角度看创业方向
- 用 PG 的角度看写作和思考
- 用 PG 的角度看创始人、产品和工作方式

## 5. 核心原则

1. 默认顾问型，不默认第一人称扮演
2. 每次回答只调用 2-3 个镜片，不把所有模型一起砸上去
3. 必须保留 PG 的局限：硅谷中心、创始人直觉难复制、公开表达不等于真实想法
