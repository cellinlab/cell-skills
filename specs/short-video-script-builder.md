# 短视频脚本工坊 Skill 设计文档

## 1. 目标概述

这个 Skill 用于把中文短视频脚本创作，从“凭感觉模仿爆款”，升级为“对标拆解 + 公式提炼 + 脚本分镜生成 + 合规检查”的系统工作流。

它主要回答：

- 一个短视频脚本到底该怎么拆
- 没有竞品时，能不能先用经过验证的结构起草
- 有竞品时，应该提炼什么，而不是只抄表面句子
- 前 3 秒、分镜、节奏、CTA、平台合规应该如何一起考虑

## 2. 设计来源

核心吸收 `huashu-douyin-script`，但做 Cell 风格改造。

保留的部分：

- benchmark-first 的脚本工作流
- 7 维分析框架
- “前 3 秒 = 文案钩子 + 视觉钩子”的硬规则
- 种草 / 投放两类脚本差异
- 公式沉淀与合规检查

改造的部分：

- 从“抖音电商专用”扩成“中文短视频脚本工坊”，兼容种草、教程、口播、评测、投放
- 不强绑定特定脚本工具环境
- 更强调结构抽象与可迁移性，而不是照抄带货口吻

## 3. 技能定位

它是一个“短视频脚本与分镜工作流” Skill。

它不是：

- 任意视频文案生成器
- 只会写带货稿的模板机
- 替代真实拍摄理解的万能拆解器

## 4. 核心原则

1. 先拆 benchmark，再写脚本
2. 没有 benchmark 时，使用公式库做基线，但要明说置信度较低
3. 脚本要同时考虑钩子、视觉、节奏、转化和合规
