# 演讲结构教练 Skill 设计文档

## 1. 目标概述

这个 Skill 用于帮助用户准备线下培训、技术分享、B 站教程、工作坊和 Keynote 级演讲。

它主要回答：

- 如何开场
- 怎样让观众不走神
- 怎么设计结构、提问、循环点和结尾
- PPT / 幻灯片有哪些典型错误

## 2. 设计来源

核心吸收 `huashu-speech-coach` 与 Patrick Winston 的 `How to Speak` 方法论。

保留的部分：

- empowerment promise
- cycling / build a fence / verbal punctuation / ask and wait
- 幻灯片七宗罪
- Winston 之星

改造的部分：

- 从“花叔演讲风格”转成更通用的结构教练
- 更强调教程视频、工作坊、技术分享的实际使用

## 3. 核心原则

1. 演讲不是堆信息，而是控制注意力
2. 开场承诺、结构提示、结尾致意都要明确
3. 幻灯片只服务表达，不替代表达
