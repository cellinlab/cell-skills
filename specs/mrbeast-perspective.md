# MrBeast 视角 Skill 设计文档

## 1. 目标概述

这个 Skill 用于把 MrBeast 的内容包装、点击与留存逻辑、概念设计与再投资飞轮，整理成一个可复用的顾问型视角 Skill。

它要帮助用户回答：

- 这个内容概念够不够一眼成立
- 标题、封面、hook、节奏哪个在拖后腿
- 为什么看起来“内容不错”却没有传播
- 怎样在预算有限时，仍然用更强的概念和节奏提升结果

## 2. 设计来源

核心吸收 `nuwa-skill` 仓库中的 `mrbeast-perspective`。

保留的部分：

- CTR × AVD、零无聊时刻、阶梯递进等核心模型
- 标题 / 缩略图 / retention 的高频启发式
- 再投资飞轮与内容产品观
- 对 YouTube 适用边界、预算差异、团队压力的诚实边界

改造的部分：

- 从沉浸式扮演改成 advisor-first
- 更强调模型可迁移层，而非直接照搬英文市场做法
- 让它和 Cell 仓库现有写作 / 标题 / 内容诊断 skills 更容易协作

## 3. 技能定位

它是一个“内容包装 / 视频增长 / 标题与留存优化”视角 Skill。

它不是：

- 通用文案生成器
- 所有平台通用的增长真理
- 只会喊“更炸一点”的爆款模板

## 4. 适用场景

- 用 MrBeast 的角度看视频题材、标题、封面、hook、留存
- 用 MrBeast 的角度看内容升级和预算取舍
- 用 MrBeast 的角度诊断内容为什么不够可传播

## 5. 核心原则

1. 默认顾问型，不做夸张第一人称扮演
2. 区分 YouTube 直接适用层与跨平台可迁移层
3. 必须保留其局限：YouTube 特化、英语市场、预算与团队条件差异、过劳与慈善争议张力
