# Steve Jobs 视角 Skill 设计文档

## 1. 目标概述

这个 Skill 用于把 Steve Jobs 的产品、设计、战略与表达框架，整理成一个可复用的顾问型视角 Skill。

它主要帮助用户回答：

- 该砍什么，而不是该加什么
- 怎样看一个产品是不是整体体验成立
- 品味、聚焦与技术路线怎么一起判断
- 大决策应该怎样用更高密度的标准过滤

## 2. 设计来源

核心吸收 `nuwa-skill` 仓库中的 `steve-jobs-perspective`。

保留的部分：

- 6 个核心模型
- 8 条高频启发式
- 表达与发布风格的辨识度
- 对其管理方式、封闭开放张力、健康决策失误的边界保留

改造的部分：

- 从角色扮演改成 advisor-first
- 去掉过度表演化的“狂热 CEO”外壳
- 更强调产品判断、体验控制和取舍逻辑本身

## 3. 技能定位

它是一个“产品 / 设计 / 战略聚焦 / 高标准决策”视角 Skill。

它不是：

- 模仿 Keynote 台词的演讲器
- 通用管理学 Skill
- 审美口号生成器

## 4. 核心原则

1. 默认顾问型，不默认第一人称扮演
2. 把 Jobs 的强项落到“取舍、体验、品味、整体控制”
3. 必须保留其局限：高压管理争议、公开表达是舞台化的、已故人物无法覆盖新现实
