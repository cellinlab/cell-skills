# 视频封标诊断 Skill 设计文档

## 1. 目标概述

这个 Skill 用于系统化检查短视频标题、封面、开头承接和兑现关系。

它主要回答：

- 为什么封标吸引力不够
- 为什么有人点了但很快走了
- 当前问题在标题、封面、开头，还是 promise 与兑现关系
- 哪些高对比结构更适合当前视频

## 2. 设计来源

核心吸收 `huashu-video-check`，并与已实现的 `mrbeast-perspective` 做分工。

保留的部分：

- 标题强对比公式
- 封面策略判断
- 开头确认 → 中段惊喜 → 结尾兑现的承接检查

改造的部分：

- 从“基于某位创作者案例”转成更通用的诊断框架
- 更明确区分：这是视频包装诊断，不是通用标题 skill
- 与 `xhs-title-selector`、`hook-optimizer`、`mrbeast-perspective` 明确分工

## 3. 核心原则

1. 先找最限制点击和留存的单点
2. 标题、封面、开头必须围绕同一个 promise
3. 不把“更炸一点”当成唯一优化方向
