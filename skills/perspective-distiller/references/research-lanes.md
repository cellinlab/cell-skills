# 六路研究工作流

## 1. 信源优先级

默认信源顺序：

1. 用户提供的一手材料
2. 本人著作、演讲、访谈、长文、播客
3. 本人社交媒体或公开碎片表达
4. 权威传记、深度采访、研究文章
5. 高质量外部评论与批评
6. 低质量聚合页、百科页、营销号，只作线索，不作核心证据

知乎、微信公众号、论坛搬运、短摘要页如果没有原始链接，最多当 lead，不当核心证据。

## 2. 六条研究线

### Lane 1: Writings

看：

- 书
- 长文
- 论文
- newsletter
- 系统性公开写作

重点：

- 反复出现的核心论点
- 自创术语
- 推荐书单或思想谱系

### Lane 2: Conversations

看：

- 播客
- AMA
- 长访谈
- 对谈
- Q&A

重点：

- 被追问时怎么展开
- 即兴类比
- 什么时候承认不知道
- 遇到反驳如何应对

### Lane 3: Expression DNA

看：

- 社交媒体
- 短演讲
- 公开碎片表达

重点：

- 高频句式
- 高频词
- 幽默与锋利方式
- 断言 / 犹豫比例

### Lane 4: External Views

看：

- 他人评价
- 批评者视角
- 传记
- 同行比较

重点：

- 别人反复观察到的模式
- 典型争议
- 外部看见但本人未明说的结构

### Lane 5: Decision Records

看：

- 关键决策
- 产品转折
- 投资、组织、公开立场变化

重点：

- 他在真实压力下怎么选
- 言行一致还是不一致

### Lane 6: Timeline and Recent Updates

看：

- 人生阶段
- 关键节点
- 最近 12-24 个月的动态

重点：

- 思想是否有演化
- 哪些材料过时了

## 3. 本地素材优先模式

如果用户提供了：

- PDF
- transcript
- 字幕
- 文章存档
- 私人整理笔记

先读这些，再判断哪些 lane 仍需网络补足。

## 4. 研究文件建议

默认在目标 Skill 下保存：

```text
references/research/
├── 01-writings.md
├── 02-conversations.md
├── 03-expression-dna.md
├── 04-external-views.md
├── 05-decisions.md
└── 06-timeline.md
```

每个文件至少写明：

- 关键发现
- 来源
- 置信度
- 和其他 lane 的对应关系
