# 思维框架提炼方法

## 1. 五层提炼对象

蒸馏时默认提取五层：

1. **心智模型**
   - 他反复用什么镜片看世界
2. **决策启发式**
   - 他遇到问题时常用的判断捷径
3. **表达 DNA**
   - 他怎么组织句子、转折、类比、犹豫和断言
4. **反模式 / 价值底线**
   - 他明确不做什么、讨厌什么、拒绝什么
5. **诚实边界**
   - 有哪些东西根本蒸不出来，或材料不足以支撑

不要把五层混成一锅。

## 2. 三重验证

一个观点要进入“核心心智模型”，默认至少通过下面三项中的两项；三项全过最稳。

### 2.1 跨域复现

同一个框架至少在两个不同主题或场景里出现过。

如果只在一个访谈里提过一次，更可能是临场说法，而不是稳定镜片。

### 2.2 生成力

用这个框架，能推断他面对新问题的大致立场或问题切法。

如果一个观点无法迁移到新场景，它更像一句结论，不像一个模型。

### 2.3 排他性

这不是“所有聪明人都会说的话”，而是这个对象比较独特的视角。

越有辨识度，越值得被纳入核心模型。

## 3. 分级规则

- 通过 3 项：核心心智模型
- 通过 2 项：强启发式，可放进模型或 heuristics，视材料强度决定
- 通过 1 项：保留为启发式，不升级
- 通过 0 项：通常不纳入

## 4. 表达 DNA 提炼

优先观察这些维度：

- 句子长短
- 结论先行还是铺垫先行
- 类比密度
- 疑问句比例
- 确定性 / 犹豫性
- 高频词与禁忌词
- 幽默方式

目标不是滑稽模仿，而是抓住“辨识度最高的骨架”。

## 5. 矛盾处理

矛盾通常有三类：

### 5.1 时间性演化

早年说法与近期说法不同。

处理：

- 标出时间
- 默认以近期稳定观点为主
- 记录演化，不要抹平

### 5.2 场景性差异

在不同领域或不同角色下，判断标准不同。

处理：

- 保留场景条件
- 不强求统一口径

### 5.3 核心张力

比如既强调自由又强调纪律，既追求极致又承认现实约束。

处理：

- 明确写成 tension
- 这往往是最有价值的地方

## 6. 置信度

建议在研究文件里用以下标记：

- `A`：一手、反复出现、可交叉验证
- `B`：一手但覆盖不足，或二手高质量交叉支持
- `C`：弱二手、推断、待验证

最终 Skill 里尽量少放 `C` 级内容。

## 7. 人物 Skill 与主题 Skill 的差别

| 维度 | 人物 | 主题 |
| --- | --- | --- |
| 核心 | 一个人的认知系统 | 一个领域的方法工具箱 |
| 表达 | 更有个人辨识度 | 更偏中性或多流派 |
| 张力 | 个人张力 | 流派分歧 |
| 验证 | 对照此人公开材料 | 对照领域共识与主要流派 |
